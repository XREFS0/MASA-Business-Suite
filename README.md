# MASA Business Suite 🚀
### *The Ultimate Premium & All-in-One Enterprise Resource Planning (ERP) Solution*

**MASA Business Suite** (MASA ERP Pro) is a modern, high-performance, and feature-rich desktop ERP application written in C# WinForms. It features a stunning, borderless, user-friendly UI/UX built with premium Guna UI components and SQLite database, offering lightweight, self-contained business management tools for enterprise operations.

---

## 🌟 Visual Preview & Aesthetics
The application features a beautifully polished dark and light mode theme, high-resolution animations, custom-drawn charts, and a state-of-the-art animated splash screen built to mimic premium modern SaaS platforms.

---

## 🛠️ Key Features

### 1. 📊 Modern Dashboard
* **Dynamic Analytics**: Real-time sales curves and financial charts using smooth native Bezier curves.
* **KPI Metric Cards**: Quick-glance summaries of total products, active employees, and net profits with visual growth indicators.
* **Activity & Logs Tracker**: Real-time timeline listing recent activities, errors, and logins.

### 2. 👥 Human Resources (HR) & Operations
* **Employee Directory**: Profile tracking including department, contact info, join dates, and salary.
* **Department Management**: Organizational structure builder assigning managers and descriptions.
* **Attendance System**: Live check-in/check-out logs, automatically tracking late arrivals and absentees.
* **Payroll Processing**: Automatic generation of net salary payments factoring in bonuses and deductions.

### 3. 🛍️ Inventory & Warehousing
* **Product Catalog**: Barcode management, purchase price, selling price, and minimum stock alerts.
* **Categories**: Clean taxonomy manager for grouping assets.
* **Stock Controls**: Real-time stock counts.

### 4. 🛒 Point of Sale (POS) & Sales
* **Cashier Checkout**: Clean, modern POS interface with direct invoice generation and instant calculations.
* **Invoice Database**: List of all finalized invoices, statuses, and customer names.
* **Customer Loyalty**: Loyalty points tracker rewarding customers based on purchase behavior.

### 5. 💰 Finance & Expense Management
* **Expense Log**: Tagged business expenses (rent, utility, maintenance) with transaction dates.
* **Net Profits**: Automatic financial calculators comparing sales revenue against expenses.

### 6. 🎨 Settings & Customization
* **Universal Themes**: Sleek real-time toggle between Light Mode and Dark Mode.
* **Global Configurations**: Company name, logo path, contact details, tax rates, and backup operations.

---

## 💻 Technology Stack
* **Framework**: .NET Framework 4.7.2 (C#)
* **UI Controls**: [Guna.UI2.WinForms](https://github.com/GunaOpenSource/Guna.UI2) (Premium modern WinForms controls)
* **Database**: SQLite (Compact, embedded, zero-configuration)
* **Design Engine**: Custom Guna theme coordinator (`ThemeManager`) ensuring perfect UI consistency.

## 🚀 Getting Started & Build Instructions

### Prerequisites
* **Visual Studio 2022** (or 2019) with the `.NET Desktop Development` workload installed.
* **.NET Framework 4.7.2** Developer Pack.

### How to Compile and Run
1. Clone or download the repository to your local system.
2. Open the solution file `MASA Business Suite.sln` inside Visual Studio.
3. Restore the NuGet packages (`System.Data.SQLite` and `Guna.UI2.WinForms`).
4. Ensure the build configuration is set to **x86** or **x64** (SQLite native DLL requires target architecture matching to load `SQLite.Interop.dll` correctly).
5. Build and run the project!
